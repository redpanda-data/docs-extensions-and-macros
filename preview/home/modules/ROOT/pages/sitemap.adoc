== Key Documentation Features

=== AI-Optimized Formats

* **MCP Server**: Interactive documentation server at https://docs.redpanda.com/mcp
* **llms.txt**: Curated overview following llms.txt standard
* **llms-full.txt**: Complete documentation export
* **Component exports**: ROOT-full.txt, cloud-full.txt, redpanda-connect-full.txt
* **Markdown pages**: Every HTML page has .md equivalent

=== Metadata Standards

All documentation pages include:

* **Git dates**: Accurate created/modified dates from git history
* **Version information**: Component version, product versions
* **Topic type**: Tutorial, how-to, concept, reference
* **Learning objectives**: Measurable outcomes for tutorials
* **Personas**: Target audience (developer, operator, architect)
* **Categories**: Content categorization
* **Schema.org markup**: TechArticle and FAQPage structured data

=== Version Access Patterns

**Self-Managed (ROOT)**:
[source]
----
/current/path           # Latest stable version
/25.3/path             # Specific version
/24.3/path             # Older version
----

**Cloud**:
[source]
----
/cloud/current/path    # Latest (only version)
----

**Connect**:
[source]
----
/redpanda-connect/current/path  # Latest (only version)
----
